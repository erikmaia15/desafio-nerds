para rodar a aplicação clone esse repositório, entre na pasta, pelo terminal "cd desafio-nerds" ou pela interface da sua IDE, quando entrar na pasta, digite "npm install" para baixar as depêndencias, rode a API mockada abrindo um terminal e digitando esse comando "npm install -g json-server", deixe rodando no terminal, agora abra outro terminal, e digite npm run dev, para rodar o projeto, e entre na url que vai aparecer no seu terminar, normalmente é "http://localhost:5173/", no navegador de sua preferência.
